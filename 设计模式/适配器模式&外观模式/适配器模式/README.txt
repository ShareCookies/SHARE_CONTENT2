适配器模式
	定义：
		将一个类的接口,转换成客户期望的另一个接口。
		适配器让原本接口不兼容的类可以合作无间。
	实现思路：
		适配器模式实现的核心点是适配器的创建：
			适配器实现目标接口。
			然后把适配对象的行为，适配为目标接口的行为。
	例：
		编写适配器：
			//适配器模式关键点
			//适配器实现目标接口。然后把适配对象的行为，适配为目标接口的行为。
			./例/TurkeyAdapter.java
		该适配器作用是用火鸡冒充鸭子：
			适配对象：火鸡 ./例/WildTurkey.java
			目标对象：鸭子 ./例/Duck.java
		测试入口：
			./例/AdapterTest.java
