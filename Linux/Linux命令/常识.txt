linux注意点：
	Linux有大小写之分
	查找（）要在前头加上转义符\
Linux的发行版:
	发行版就是将Linux内核与应用软件做一个打包。
	一般来说linux系统基本上分两大类： 
		1.RedHat系列：Redhat、Centos、Fedora等 
		2.Debian系列：Debian、Ubuntu等 
Linux程序:
	cd, mkdir等都是系统自带的程序，一般在/bin或/usr/bin目录下。
快捷键：
	tab键会自动补全文件名
字符含义: 
	~ :是home目录的意思
		以超级用户root账号登陆：~ 是 /root目录
		以普通用户name登陆：~是 /home/name目录
	/:
		根目录
		Linux最顶层只有一个根目录，所有文件都存储在这一个根目录之下。
	|:管道
		把前一个命令的结果当作，|后命令的输入
		例：
			ps -ef|grep java	
			ps -ef 查找所有程序的运行情况
			grep java 搜索给定的文件，得到与给定模式匹配的行
	?	
		//代表任意一个字符，如man？，那么man1就可以被其匹配到
	> 输出重定向：
		> 是定向输出到文件：
		如果文件不存在，就创建文件；
		如果文件存在，就将其清空；
		一般我们备份清理日志文件的时候，就是这种方法：先备份日志，再用`>`，将日志文件清空（文件大小变成0字节）；
	>> 这个是将输出内容追加到目标文件中：
		如果文件不存在，就创建文件；
		如果文件存在，则将新的内容追加到那个文件的末尾，该文件中的原有内容不受影响。
	--和-：
		两个后面一般都是跟参数选项
		--后面一般跟长的完整名字 如 chkconfig --add
		-后面一般跟简写大多数是一个字母，如 ls -l
帮助文档:
	man、info、help,cheat!!!
	man：（manual）
		man 命令显示系统的手册页,在/usr/share/man下，如果系统没有文档，是不会显示这些帮助信息的。
		比help要详细。它可以用来查看命令，函数，
	help:
		命令--help 选项，用来显示工具的一些信息，用法
	info:
		包括一些关于Linux Shell，工具，GNU项目开发程序说明文档,不但显示一些选项，还会有一些范例
		注:一般man用的比较多，比help细，比info简单
