开发工具使用eslint：
	idea使用Eslint格式化js：
		如果你的项目中做了eslint的配置，js文件时鼠标右击，会多一个Fix Eslint Problems的选项；
		该选项可以根据eslint规则自动修复js文件中出现的常见错误。
		该选项本质上就是执行了eslint --fix filename.js。
	idea安装eslint：
		https://juejin.im/post/5c668291e51d457832120627
		idea启用Eslint
			Languages & Frameworks -> JavaScript -> Code Quality Tools -> ESLint
		注：
			idea每开启一个前端项目，均要重新开启eslint的支持。
