JavaIO
介绍:
Java中使用IO来进行数据读取和写入。
JavaIO作用:
Java I/O系统用于各种IO源端和接收端(文件、控制台、网络链接等)之间的通信,而且还需以多种不同的方式通信(顺序、随机存取、缓冲、二进制、按字符.按行、按字等)。
Java使用大量的类解决通信问题。
File类:
实用类库工具，能处理文件或文件集（文件夹）。
输入/输出:
介绍:
Java中使用流这个抽象概念，待表任何能出数据的数据源对象或能接收数据的接收端对象。
流屏蔽了具体的I/O设备处理数据的细节。
		数据源包括：
1.字节数组
2.string对象
3.文件
4."管道",从一端输入从另一端输出。
5.一个由其他种类的流组成的序列，一边我们把它们收集合并到一个流中。？6.其他数据源，如Internet连接等。
		输出流去往目标(接收端对象)：
			字节数组，文件或管道。
划分：
根据数据的走向可分为输入流和输出流。
	注：这个走向是以jvm内存为基准的，即往jvm内存中读数据是输入流，从jvm内存中往外写是输出流。
