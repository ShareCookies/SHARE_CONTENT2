解释:
	超类(SuperClass) ：
		被继承的类称为超类(SuperClass)，也叫做父类。
	类方法:
		用static修饰，也叫静态方法
	实例方法:
		没有用static修饰，也叫非静态方法

形参为何要为final:?
	简单理解就是，拷贝引用，为了避免引用值发生改变，例如被外部类的方法修改等，而导致内部类得到的值不一致，于是用final来让该引用不可改变。
构造函数：
	https://www.cnblogs.com/zerahmu/p/7662810.html
	子类被初始化的时候，会在构造函数中先调用父类的构造函数。
	注：
		可无需手动调用，java默认使用super();调用父类的无参构造函数。
