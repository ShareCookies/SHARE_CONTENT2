命令行运行java文件
	1. 编写Hello.java 文件
	2. 进入对应目录，输入 javac Hello.java 编译程序。
		//编译成功后，系统生成 Hello.class 文件默认存放在与 Hello.java 同目录中。
		//文件要迁移到对应包名目录下，com/test/Hello.class
	3. 继续编译页面输入“java 包名.Hello”运行 Java 程序
